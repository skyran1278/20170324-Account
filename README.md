# 2017-frontend-starter-kit

NodeJS/NPM/ES6/Bower/Gulp/Sass/Babel/Babelify/Browserify

## Usage

```
Install Git NodeJS
```

```
$ git clone https://github.com/skyran1278/2017-frontend-starter-kit.git
```

```
$ npm install && bower install
```

```
$ npm start
```

```
localhost:7777
```
